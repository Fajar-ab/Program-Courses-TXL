#### 4. Perhatikan 3 struktur deklarasi variabel berikut:
I
```c++
int a, b, c;
int d, e;
```
II
```c++
int a, b, c;
int A, B;
```
III
```c++
int a, b, c;
double A, B;
```
Di antara ketiga struktur tersebut, pendeklarasian variabel yang tepat adalah ....

- [ ] &nbsp; __a__ &nbsp; I
- [ ] &nbsp; __b__ &nbsp; II
- [ ] &nbsp; __c__ &nbsp; III
- [ ] &nbsp; __d__ &nbsp; I dan III
- [x] &nbsp; __e__ &nbsp; I, II, dan III
